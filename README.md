# Swiggy Clone Backend

Enterprise-style Food Delivery Backend inspired by Swiggy, developed using Java, Spring Boot, Spring Security, Spring Data JPA, MySQL, REST APIs, and Postman. The application follows a layered architecture approach and supports end-to-end food ordering and delivery workflows.

## Features

- User Registration and Management
- Restaurant Management
- Food Item Management
- Order Placement and Processing
- Delivery Partner Assignment
- Real-time Order Status Tracking
- Authentication and Authorization using Spring Security
- RESTful API Development
- MySQL Database Integration
- Exception Handling
- API Testing with Postman

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven
- Postman
- GitHub

## Architecture

Controller Layer → Service Layer → Repository Layer → MySQL Database

## API Modules

- User APIs
- Restaurant APIs
- Food Item APIs
- Order APIs
- Delivery Partner APIs

## Key Highlights

- Implemented entity relationships using JPA annotations.
- Managed order lifecycle from PLACED to OUT_FOR_DELIVERY and DELIVERED states.
- Integrated Spring Security for secure API access.
- Tested and validated APIs using Postman.
- Maintained source code using GitHub for version control and collaboration.
