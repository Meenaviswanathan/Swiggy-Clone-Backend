# Swiggy Clone Backend

Enterprise-style Food Delivery Backend inspired by Swiggy, developed using Java, Spring Boot, Spring Security, MySQL, JPA/Hibernate, REST APIs, and Postman.

## Features
- User Management
- Restaurant Management
- Food Item Management
- Order Placement
- Delivery Partner Assignment
- Order Status Tracking
- Authentication and Authorization
- RESTful APIs
- MySQL Database Integration

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman
